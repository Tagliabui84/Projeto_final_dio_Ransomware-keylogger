## 🔬 Principais Aprendizados

### Ransomware:
- Utiliza criptografia assimétrica/simétrica para sequestro
- Ataque tipicamente começa por phishing ou RDP exposto
- Backups offline são a defesa mais efetiva

### Keylogger:
- Pode ser implementado em poucas linhas de código
- Técnicas furtivas dificultam detecção
- MFA protege mesmo com senhas comprometidas

## 🛡️ Recomendações de Segurança

1. **Prioridade #1**: Implementar backup 3-2-1
2. **Prioridade #2**: Ativar MFA em todos os serviços
3. **Prioridade #3**: Treinamento de conscientização
4. **Prioridade #4**: Implementar EDR/antivírus moderno
